== Description ==

**Caution**

This plugin requires PHP 5.3 or later.


I was tired or customize the functions.php of the theme in the same plugin to install every time.
To which functionality needs to be minimal, often, though it together if you do. From the idea and made this plugin.

This plugin functionality itself is not much, minimum required features are stuffed one plugin.

**Features:**

* Basic settings
  * Disable xml-rpc.
  * Disable author page.
  * Disable wp generator tag.
  * Enhance feed links.
  * Enhance short link.
  * Disable wpautop.


* Original Dashboard Widget

* Database optimization

* Ace Editor

* Breadcrumbs

* Admin Menu Editor

* Admin Bar Template

* Admin Post Navigation

* Other Tools

Inform to the github repository for the following if you request something or fix.

[Github] (https://github.com/1shiharaT/wp-assistant/)

== Installation ==

1. Upload `wp-assistant` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
